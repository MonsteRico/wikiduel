# Model Infoboxes as semantic Article Document content

Wiki Duel represents a Wikipedia Infobox as a dedicated semantic Article
Document block rendered by a dedicated production component, not as generic
table data or sanitized upstream HTML. This preserves useful facts, attributed
images, and canonical Navigation Nodes while keeping heterogeneous table layout,
unsupported controls, and executable markup outside the client contract; generic
data-table support remains a separate capability.
