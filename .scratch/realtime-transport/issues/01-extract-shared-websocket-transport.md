# Extract the Shared WebSocket Transport

Status: completed
Scope: MVP required
Category: enhancement

## What to build

Extract the browser's physical WebSocket connection from Lobby-specific state into one application-level transport that future realtime features can share. The transport owns connection lifecycle, connection status, JSON envelope decoding, typed sending, and message subscription or dispatch; Lobby behavior remains a feature-specific consumer with no visible behavior change.

- [x] Listener cleanup prevents duplicate delivery after unmounting or route changes.
