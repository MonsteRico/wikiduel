# Expand the Prompt Pool

Status: needs-triage
Scope: MVP optional
Category: enhancement

## What to build

Expand the manually tested prompt pool from ten to between 25 and 50 enabled prompts before inviting a larger test group, using early playtest observations to improve category and difficulty coverage.

## Acceptance criteria

- [ ] At least 25 and no more than 50 enabled prompts pass the same endpoint and playability checks as the first ten.
- [ ] The pool has intentional subject and rough-difficulty variety.
- [ ] Known trivial, broken, or frustrating prompts are disabled or replaced.

## Blocked by

- [`prompt-pool/01`](./01-curate-first-ten-prompts.md)
- Feedback from the first small-group test.

## Out of scope

- Automatic prompt generation
- Algorithmic difficulty or shortest-path scoring
