# Generate Prompt Candidates

Status: needs-triage
Scope: Future
Category: enhancement

## What to build

Explore an evidence-led way to generate candidate ordered Prompts for later human review without allowing generated content to enter the enabled production pool automatically.

## Acceptance criteria

- [ ] Generated candidates conform to the implemented Prompt format.
- [ ] Generation never enables, publishes, or replaces a production Prompt without explicit human review.
- [ ] The future ticket is refined against observed manual curation cost and quality requirements before implementation.

## Blocked by

- Evidence that maintaining the Prompt pool manually is insufficient.

## Out of scope

- Automatic production Prompt publication
- Replacing human validation, difficulty assignment, or quality judgment
