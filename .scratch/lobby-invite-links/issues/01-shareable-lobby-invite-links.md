# Add Shareable Lobby Invite Links

Status: needs-triage
Scope: MVP optional
Category: enhancement

## What to build

Let a Host copy a URL that carries the five-character Lobby code and gives the recipient a clear confirmation step before joining as the opponent.

## Acceptance criteria

- [ ] Opening a valid invite presents the correct Lobby code before claiming its seat.
- [ ] Invalid, expired, and full Lobbies produce clear outcomes.
- [ ] Sharing a URL does not expose Player Session credentials.

## Blocked by

- The required private Lobby flow.

## Out of scope

- Universal or native-app deep links
- Public discovery or matchmaking

- 2026-07-03: Classified as MVP optional for a larger-group test; manual code sharing is sufficient initially.
