# Complete an Accessibility Fundamentals Pass

Status: needs-triage
Scope: MVP optional
Category: enhancement

## What to build

Before widening the test group, make core Lobby, Duel, comparison, and summary flows keyboard operable with visible focus, semantic article structure, reduced-motion behavior, and status cues that do not rely on color alone.

## Acceptance criteria

- [ ] Every required gameplay action is reachable and operable by keyboard.
- [ ] Focus remains visible and moves predictably across screen transitions and overlays.
- [ ] Playable Article headings, prose, lists, links, images, and captions retain useful semantics.
- [ ] Motion honors the user's reduced-motion preference.
- [ ] Player, connection, outcome, and readiness states remain understandable without color.

## Blocked by

- Stable required-MVP screens and interactions.
